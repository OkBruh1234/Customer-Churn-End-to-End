# 📊 Customer Churn Prediction System

A work on Diabetes Prediction by analyzing 7,300 customers on different criterias like contarct, tenure time and tenure_bucket to further divide the categories into segments and deployed this in complete end to end fashion

## 🚀 Overview
End-to-end ML system to predict customer churn using Random Forest.

## 🔍 Features
- Data analysis & feature engineering
- ML models: Logistic, RF, XGBoost
- Threshold optimization for recall
- Business insights (tenure & contract)
- Streamlit deployment

## ⚙️ Tech Stack
- Python
- Scikit-learn
- Streamlit

## 🎯 Output
- Churn probability
- Risk classification
- Explanation of prediction

## 📈 Key Insight
Customers with low tenure and month-to-month contracts have highest churn risk.
